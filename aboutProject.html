About this Project

This project is a web-based sales dashboard developed using HTML, CSS, and JavaScript, with Chart.js powering the interactive charts. It aims to visualize sales data clearly and intuitively, helping users explore trends, compare regions, and analyze product performance through various types of charts such as bar, line, and pie charts.

The dashboard includes filtering features that let users select date ranges and view data breakdowns by region or product category. Although this project uses static sample data, itâ€™s structured to be easily extended with real-time data sources or backend integration.
